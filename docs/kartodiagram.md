# Kartodiagram
Podstatou metody **kartodiagramu** (angl. diagram map) v nejtypičtější podobě je **znázornění struktury a absolutní hodnoty jevu** pomocí diagramů proměnlivé velikosti. Společně s hodnotou jevu bývá často znázorňována také vnitřní struktura jevu. Kartodiagram pracuje s kvantitativními daty, která se typicky týkají územních celků (diagram se vztahuje k ploše), jednotlivých míst (diagram se vztahuje k bodu) či liniových prvků. [*(Lysák, 2014)*](https://drive.google.com/file/d/1q4zpn4ZXQr7XSeBQT3GEvf9cmv6tOsEx/view)
